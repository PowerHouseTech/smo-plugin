== Description ==

The Thank-You Popup Generator plugin is an extension of WooCommerce that adds a popup window after checkout, giving one last opportunity to enhance the customer's shopping experience. The popup allows for a customized thank-you message, along with a pleasing interface and an inspirational quotation. The popup loads after the customer has submitted their order and before (overlaying) the order confirmation page.


== Installation ==

1. Upload `thank-you-popup-generator.php` to the `/wp-content/plugins/` directory or install it in the WordPress admin area
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The admin area for the plugin is located within the WordPress Settings area.
